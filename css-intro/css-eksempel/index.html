Velkommen til CSS

Hvor fargelegging blir gøy igjen!

CSS er et stilsett-språk som brukes for å kontrollere utseendet og layouten til nettsider. Det hjelper utviklere med å gjøre nettsider både brukervennlige og visuelt tiltalende.

Hva bruker vi det til?

  • Få det som synes på skjermen til å se fint ut
  • Skape responsivt webdesign som tilpasser seg ulike skjermstørrelser
  • Øke lesbarheten av innholdet på nettsider
  • Gjøre nettsider mer tilgjengelige for ulike brukergrupper, inkludert funksjonshemmede (universell utforming)
